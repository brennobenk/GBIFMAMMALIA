## 📌 Sobre

Este projeto utiliza dados abertos do GBIF para analisar e visualizar registros de ocorrência da classe **Mammalia** ao redor do mundo.

## 🚀 Como usar

1. Acesse o link acima
2. Explore os dados de ocorrência de mamíferos
3. Utilize os filtros disponíveis para refinar sua pesquisa

## 📊 Fonte dos dados

Os dados são provenientes do **GBIF — Global Biodiversity Information Facility**, uma infraestrutura internacional de dados abertos sobre biodiversidade.

- 🔗 [GBIF.org](https://www.gbif.org/)
